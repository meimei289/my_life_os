## 🗂 Structure

```
my_life_os/
├── Goals/
│   ├── vision.md          # 長期ビジョン
│   ├── roadmap.md         # 年間・四半期マイルストーン
│   ├── career/            # キャリア・転職
│   ├── language/          # 英語・IELTS
│   ├── finance/           # お金・貯金
│   ├── netherlands/       # オランダ移住
│   ├── health/            # 健康・習慣
│   └── learning/          # 読書・学習
├── Tasks/                 # 週次・日次タスク
└── Journal/               # 日次ログ・振り返り
```

---

## 🔄 運用ルール

1. **毎週日曜** — `Tasks/` を更新・先週をレビュー
2. **毎日** — `Journal/` に一言ログ
3. **四半期ごと** — `Goals/roadmap.md` を見直す
4. **マイルストーン達成時** — 各 `goal.md` の状態を更新（✅）
