# 💪 健康・習慣 アクション

_最終更新：2026-04-05_

---

## 🔄 毎日やること（維持）

- [ ] 3:50 起床
- [ ] 運動・筋トレ（週3〜5回）
- [ ] 睡眠7時間以上確保
- [ ] Journal に今日の状態を一言書く

---

## 🔄 ストレス管理（仕組みとして組み込む）

- [ ] 週に1回、「今週のストレス源」をJournalに書き出す
- [ ] 「手放すリスト」を作る（コントロール外のことを明示する）
- [ ] 転職・IELTS・貯金の進捗を週次でレビューして不安を言語化する
- [ ] 月1回、意図的に「何も予定しない半日」を作る

---

## ⬜ 転職後（新環境への適応）

- [ ] 新しい生活リズムの中でも3:50起床を維持する
- [ ] 通勤・職場環境の変化に合わせてルーティンを微調整する
- [ ] ストレスが高い時期はJournalの頻度を上げる

---

## ⬜ オランダ移住後（習慣の再構築）

- [ ] 時差・環境変化の中でも起床時間を再設定する
- [ ] 現地のジムや運動環境を早めに確保する
- [ ] 孤独・慣れない環境によるストレスを Journal で言語化する
